# 🚀 Disciplina 90D - 90 Dias de Alta Performance

Um rastreador interativo de disciplina pessoal para ajudá-lo a completar um ciclo de 90 dias de desenvolvimento pessoal com foco, consistência e transformação.

## 📋 Características

### ✅ 3 Fases de Transformação

1. **Fase 1: Fundação (Dias 1-30)**
   - Entender seus gatilhos e âncoras matinais
   - Tarefas: Acordar no horário, planejamento, deep work, modo foco

2. **Fase 2: Compromisso (Dias 31-60)**
   - Aumentar intensidade e exercício físico
   - Tarefas: Exercício, meditação, foco profundo, ritual noturno

3. **Fase 3: Identidade (Dias 61-90)**
   - Agir como quem você se tornou
   - Tarefas: Prioridades semanais, revisão diária, voto de identidade

### 📊 Recursos Principais

- ✅ **Checklist Diário**: Tarefas específicas para cada fase
- 📝 **Reflexão Rápida**: Espaço para anotações e aprendizados diários
- 📈 **Rastreamento de Progresso**: Visualize seu avanço com barra de progresso
- 💾 **Armazenamento Local**: Todos os dados salvos no seu navegador
- 📊 **Estatísticas**: Dias completos e taxa de conclusão
- 💡 **Dicas Diárias**: Sugestões variadas para melhorar seu desempenho
- 📥 **Exportar Dados**: Faça backup do seu progresso em JSON

## 🚀 Como Usar

1. **Abra o site** em seu navegador
2. **Marque as tarefas** conforme as completa durante o dia
3. **Escreva uma reflexão** antes de dormir
4. **Acompanhe seu progresso** na barra de progresso
5. **Respeite a regra de ouro**: "Falhar 1 dia é humano; falhar 2 é uma escolha"

## 🎯 Dicas para Sucesso

1. **Consistência**: Complete as tarefas diárias, mesmo quando não se sente motivado
2. **Reflexão**: Use o espaço de reflexão para aprender com cada dia
3. **Honestidade**: Marque apenas as tarefas que realmente completou
4. **Backup**: Exporte seus dados regularmente
5. **Compartilhe**: Inspire outros a começar seu próprio ciclo de 90 dias
